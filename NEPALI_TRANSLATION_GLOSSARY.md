# Complete Glossary of Nepali Translation Changes

This table represents the final approved vocabulary for the app. These changes should be applied globally across Nepali app copy.

## 1. Core Religious & Biblical Terms

| Original/Current Term   | Preferred Nepali Christian | Context / Reasoning                                          |
| ----------------------- | -------------------------- | ------------------------------------------------------------ |
| **धर्मशास्त्र**         | **पवित्र शास्त्र**         | Standard term for "Holy Scripture" in Nepali churches.       |
| **मुक्ति / उद्धार**     | **मुक्ति**                 | Use _Mukti_ consistently for salvation.                      |
| **चेलाहट**              | **चेलापन**                 | Replaces the Hindi-sounding construction with proper Nepali. |
| **फसल**                 | **कटनी**                   | Biblical metaphor for "The Harvest" (Matthew 9:37).          |
| **भेला**                | **सङ्गति**                 | Natural term for Christian fellowship/gathering.             |
| **नेता**                | **अगुवा**                  | _Neta_ is political; _Aguwa_ is for church leaders/elders.   |
| **चर्च**                | **मण्डली**                 | The local church or body of believers.                       |
| **माछा मार्ने / मछुवा** | **जलारी**                  | Dignified scriptural term for "fishers of men."              |
| **गुणन**                | **वृद्धि**                 | Refers to spiritual growth/increase rather than math.        |

## 2. Divine Attributes & Scriptural Phrasing

| Original/Current Phrase         | Corrected Phrase                    | Note                                                     |
| ------------------------------- | ----------------------------------- | -------------------------------------------------------- |
| **परमेश्वर प्रेम हो**           | **परमेश्वर प्रेम हुनुहुन्छ**        | Added mandatory honorific for God.                       |
| **चिन्ता उहाँमाथि छोड्नुहोस्**  | **फिक्रीहरू उहाँमाथि राखिदिनुहोस्** | Matches the phrasing of 1 Peter 5:7 in the Nepali Bible. |
| **प्रभु मेरो गोठाला हुनुहुन्छ** | **परमप्रभु मेरो गोठालो हुनुहुन्छ**  | Standard Psalm 23 phrasing.                              |

## 3. App UI & Technical Terms

| Original/Current Term          | Preferred Nepali   | Note                                          |
| ------------------------------ | ------------------ | --------------------------------------------- |
| **सेटिङहरू**                   | **सेटिङ**          | Modern UI style, dropping the plural "हरू".   |
| **बजाउनुहोस्**                 | **प्ले गर्नुहोस्** | Standard term for audio player buttons.       |
| **पूरा भएको चिन्ह लगाउनुहोस्** | **पूरा गर्नुहोस्** | Concise for mobile buttons.                   |
| **पठन गतिविधि**                | **पढाइको विवरण**   | More natural than "Reading Activity."         |
| **मेट्नुहोस्**                 | **हटाउनुहोस्**     | "Remove" is often more appropriate for lists. |
| **विवेकी / विवेकशील**          | **गोप्य**          | Used for private/discreet mode.               |
| **कन्फिगर गरिएको छैन**         | **सेट गरिएको छैन** | Less technical, clearer for the average user. |

## 4. Technical Loanword List

The following terms should remain as English loanwords written in Nepali script for clarity:

- **साइन इन** (Sign In)
- **डाउनलोड** (Download)
- **सिंक** (Sync)
- **प्रोफाइल** (Profile)
- **फन्ट** (Font)
- **थिम** (Theme)
