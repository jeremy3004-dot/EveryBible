# Plan 10-01: Companion Content Contract

## Objective

Define a reusable companion-content schema and local-first sourcing strategy so optional book modules can ship now and move to backend data later.

## Scope

- Attach modular companion content to the book experience contract.
- Seed local fixture content for priority books.
- Keep missing modules explicit so the hub can collapse empty sections cleanly.

## Out Of Scope

- Replacing Learn or Harvest content systems
- Editorial tooling
- Personalized recommendations
