# Plan 18-02: Navigation Wiring, Tab Addition, and i18n Keys

## Goal

When this plan is complete, there is a 5th "Plans" tab in the bottom navigation between Learn and More, backed by a `PlansStack` navigator that renders a placeholder `PlansHomeScreen` and a `PlanDetailScreen`. The `ReadingPlanList` and `ReadingPlanDetail` routes are removed from `LearnStack`. The "Reading Plans" card in `CourseListScreen` cross-navigates to the Plans tab. All new i18n keys are added to all 4 language files.

## Scope

This plan does:
- Create `src/navigation/PlansStack.tsx` (new file)
- Create `src/screens/plans/PlansHomeScreen.tsx` as a minimal placeholder (title + segmented control skeleton) so the tab is functional
- Create `src/screens/plans/PlanDetailScreen.tsx` that wraps the existing `ReadingPlanDetailScreen` component (thin wrapper forwarding planId prop)
- Update `src/navigation/types.ts`: add `PlansStackParamList`, add `Plans` to `RootTabParamList`, add screen props helpers, remove `ReadingPlanList`/`ReadingPlanDetail` from `LearnStackParamList`, remove old screen props
- Update `src/navigation/tabManifest.ts`: add Plans entry between Learn and More
- Update `src/navigation/tabBarVisibility.ts`: add `PlanDetail` to hidden routes
- Update `src/navigation/TabNavigator.tsx`: add Plans tab screen, extend `shouldHideNestedBibleScreen` check to include `'Plans'`
- Update `src/navigation/LearnStack.tsx`: remove `ReadingPlanList` and `ReadingPlanDetail` screen entries
- Update `src/screens/learn/CourseListScreen.tsx`: change the Reading Plans card to cross-navigate to `Plans` tab via `rootNavigationRef`
- Add all new i18n keys to `src/i18n/locales/en.ts`, `es.ts`, `ne.ts`, `hi.ts`

This plan does NOT:
- Build the full PlansHomeScreen UI (that is Plan 18-03)
- Enhance the PlanDetailScreen with cover image, related plans, etc. (that is Plan 18-04)
- Touch database migrations or service functions (that was Plan 18-01)

## Prerequisites

- Plan 18-01 must be complete (types and services must exist for imports to resolve)

**Create `src/screens/plans/PlansHomeScreen.tsx` (placeholder):**

A minimal screen with a title and 4-tab segmented control that renders placeholder text for each tab. This will be fully built out in Plan 18-03.

```typescript
import React, { useState } from 'react';
import { ScrollView, StyleSheet, Text, TouchableOpacity, View } from 'react-native';
import { SafeAreaView } from 'react-native-safe-area-context';
import { useTranslation } from 'react-i18next';
import { useTheme } from '../../contexts/ThemeContext';
import { spacing, typography, radius } from '../../design/system';

type PlanTab = 'my-plans' | 'find-plans' | 'saved' | 'completed';

export function PlansHomeScreen() {
  const { colors } = useTheme();
  const { t } = useTranslation();
  const [activeTab, setActiveTab] = useState<PlanTab>('my-plans');

  const tabs: { key: PlanTab; labelKey: string }[] = [
    { key: 'my-plans', labelKey: 'readingPlans.myPlans' },
    { key: 'find-plans', labelKey: 'readingPlans.findPlans' },
    { key: 'saved', labelKey: 'readingPlans.saved' },
    { key: 'completed', labelKey: 'readingPlans.completed' },
  ];

  return (
    <SafeAreaView style={{ flex: 1, backgroundColor: colors.background }}>
      <Text style={[typography.screenTitle, { color: colors.primaryText, padding: spacing.lg }]}>
        {t('readingPlans.title')}
      </Text>
      {/* Segmented control */}
      <ScrollView horizontal showsHorizontalScrollIndicator={false}
        contentContainerStyle={{ paddingHorizontal: spacing.lg, gap: spacing.sm }}>
        {tabs.map((tab) => (
          <TouchableOpacity
            key={tab.key}
            onPress={() => setActiveTab(tab.key)}
            style={[
              styles.tabPill,
              activeTab === tab.key
                ? { backgroundColor: colors.accentPrimary }
                : { borderColor: colors.cardBorder, borderWidth: 1 },
            ]}
          >
            <Text style={[
              styles.tabLabel,
              { color: activeTab === tab.key ? colors.cardBackground : colors.secondaryText },
            ]}>
              {t(tab.labelKey)}
            </Text>
          </TouchableOpacity>
        ))}
      </ScrollView>
      {/* Placeholder content — replaced in Plan 18-03 */}
      <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
        <Text style={{ color: colors.secondaryText }}>{activeTab}</Text>
      </View>
    </SafeAreaView>
  );
}

const styles = StyleSheet.create({
  tabPill: {
    paddingHorizontal: spacing.lg,
    paddingVertical: spacing.sm,
    borderRadius: radius.pill,
  },
  tabLabel: {
    ...typography.caption,
    fontWeight: '600',
  },
});
```
