```typescript
interface LessonBottomSheetProps {

```

**Layout (Waha-style bottom sheet):**

1. Modal with `transparent={true}`, `animationType="slide"`, `visible={visible}`.
2. Outer overlay: full screen, `backgroundColor: 'rgba(0,0,0,0.5)'`, justifyContent `flex-end`. TouchableOpacity overlay to close on backdrop tap.
3. Inner sheet: `colors.cardBackground` background, `borderTopLeftRadius: 20`, `borderTopRightRadius: 20`, paddingTop `spacing.lg`, paddingBottom `spacing.xxxl` (safe area), paddingHorizontal `layout.screenPadding`.

4. **Header row**: Circular icon (40x40, `colors.accentPrimary + '18'` background, Ionicons icon matching parent foundation — hardcode 'book-outline' since we don't have the parent icon here), title `lesson.title` in `typography.bodyStrong` + `colors.primaryText`, reference `lesson.referenceLabel` in `typography.micro` + `colors.secondaryText`. Row layout with icon left, text column flex:1.

5. **Divider**: 1px height, `colors.cardBorder`, marginVertical `spacing.md`.

6. **Action rows** — each is a TouchableOpacity row with icon (24px) + label text, height 48px, flexDirection row, alignItems center, gap `spacing.md`:
   - Share Lesson Audio: Ionicons `volume-medium-outline`, text `t('gather.shareAudio')`. onPress: `Share.share({ message: lesson.title + ' - ' + lesson.referenceLabel })` then `onClose()`.
   - Share Lesson Text: Ionicons `document-text-outline`, text `t('gather.shareText')`. Same share action.
   - Share Lesson Link: Ionicons `link-outline`, text `t('gather.shareLink')`. Same share action.
   - Download: Ionicons `download-outline`, text `t('gather.download')`. onPress: no-op for now (deferred), just close.
   - Mark Complete / Mark Incomplete: Ionicons `checkmark-circle-outline` (incomplete) or `checkmark-circle` (complete), text `t('gather.markComplete')` or `t('gather.markIncomplete')` based on `isComplete`. onPress: `onToggleComplete()` then `onClose()`.
   - Manage Bookmarks: Ionicons `bookmark-outline`, text `t('gather.manageBookmarks')`. onPress: no-op (deferred), just close.

   Each row: text in `typography.body` + `colors.primaryText`, icon in `colors.secondaryText`.

7. **Close button** at bottom: TouchableOpacity, full width, centered text `t('common.done')` in `typography.button` + `colors.accentPrimary`, paddingVertical `spacing.md`, marginTop `spacing.md`.

All styles via `StyleSheet.create()`. All colors from `useTheme()`. All text via `useTranslation()`.
