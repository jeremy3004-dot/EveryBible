**Layout structure (Waha-style):**

1. SafeAreaView with `edges={['top']}`, flex:1, background from `colors.background`

2. **Sub-tab bar** at top: Two touchable tabs "Foundations" and "Topics" (using `t('gather.foundations')` and `t('gather.topics')`). Active tab has an underline indicator in `colors.accentPrimary`. Use local state `activeTab: 'foundations' | 'topics'` defaulting to `'foundations'`. The tab bar is a horizontal View with two equal-width TouchableOpacity items. Active tab text uses `colors.primaryText` + `typography.bodyStrong`. Inactive uses `colors.secondaryText` + `typography.body`. Underline: 2px height, `colors.accentPrimary`, full width of the tab.

3. **Foundations sub-tab content** (shown when activeTab === 'foundations'):
   - ScrollView with `contentContainerStyle={{ padding: layout.screenPadding, gap: spacing.lg }}`
   - **Dismissable info banner** (shown when `!useGatherStore.infoBannerDismissed`): Card with `colors.cardBackground` background, `colors.cardBorder` border, 1px borderWidth, `radius.lg` borderRadius. Contains: info text `t('gather.infoBanner')` in `typography.body` + `colors.secondaryText`, and a close (X) button (Ionicons `close-outline`, 20px) on the top-right that calls `dismissInfoBanner()`. Padding: `layout.denseCardPadding`.
   - **Foundation cards** — map over `gatherFoundations`. Each card is a TouchableOpacity:
     - First card (index 0) gets a special highlighted style: background `colors.accentPrimary + '15'` (15% opacity tint), border `colors.accentPrimary + '40'`. A small label above: `t('gather.getStarted')` in `typography.micro` + `colors.accentPrimary`.
     - Standard card: `colors.cardBackground` background, `colors.cardBorder` border, 1px borderWidth, `radius.lg` borderRadius, padding `layout.denseCardPadding`.
     - Card layout: Row with `gap: spacing.md`, `alignItems: 'center'`.
       - Left: Circular icon container (48x48, `radius.pill`, background `colors.accentPrimary + '18'`) with Ionicons `foundation.iconName` at 24px in `colors.accentPrimary`.
       - Center (flex:1): Foundation number in `typography.micro` + `colors.secondaryText` (text: `Foundations ${foundation.number}`), title in `typography.bodyStrong` + `colors.primaryText`.
       - Right: Progress text `${completedCount}/${foundation.lessons.length}` in `typography.label` + `colors.secondaryText`. Use `useGatherStore.getCompletedCount(foundation.id)` for completedCount.
     - onPress: `navigation.navigate('FoundationDetail', { foundationId: foundation.id })`

4. **Topics sub-tab content** (shown when activeTab === 'topics'):
   - ScrollView with `contentContainerStyle={{ padding: layout.screenPadding, gap: spacing.xl }}`
   - For each category in `gatherTopicCategories`:
     - Category header: `category.name` in `typography.cardTitle` + `colors.primaryText`, marginBottom `spacing.sm`
     - 2-column grid of topic cards using `flexDirection: 'row', flexWrap: 'wrap', gap: spacing.sm`. Each card takes `width: (screenWidth - 2 * layout.screenPadding - spacing.sm) / 2`.
     - Each topic card: TouchableOpacity, `colors.cardBackground` background, `colors.cardBorder` border, 1px borderWidth, `radius.lg` borderRadius, padding `spacing.md`, alignItems `center`, gap `spacing.sm`.
       - Circular icon (40x40, `radius.pill`, background `colors.accentPrimary + '18'`) with Ionicons `topic.iconName` at 20px in `colors.accentPrimary`.
       - Title: `topic.title` in `typography.label` + `colors.primaryText`, textAlign center.
       - Progress: `${completedCount}/${topic.lessonCount}` in `typography.micro` + `colors.secondaryText`.
     - onPress: `navigation.navigate('FoundationDetail', { foundationId: topic.id })` (reuse same screen — Plan 03 will handle both)

Use `useWindowDimensions` from react-native for screen width calculation. Use `useNavigation<NativeStackNavigationProp<LearnStackParamList, 'GatherHome'>>()` for typed navigation.

All styles via `StyleSheet.create()` at bottom. All colors from `useTheme()`. All strings via `useTranslation()`. Import design tokens from `../../design/system`.
