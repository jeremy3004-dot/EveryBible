# Plan 07-03: Premium Read Mode Continuity

## Objective

Refine read mode so it feels calmer and more premium while staying continuous with listen mode.

## Scope

- Tighten read-mode header controls so typography and translation tools do not crowd the text.
- Carry the active verse focus and chapter state across mode switches.
- Make the listen/read transition intentional instead of feeling like separate screens.
- Keep the existing text reader behavior accessible for chapters that already support focus verse deep links.

## Out Of Scope

- New book metadata content
- Saved content infrastructure
- Devotional, figure, and playlist modules
