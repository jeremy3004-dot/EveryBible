# Plan 09-01: Local-First Library And Queue Models

## Objective

Add local-first saved-library, playlist, and queue/history models that can power Bible listening features without waiting for backend schema work.

## Scope

- Create deterministic helpers for favorites, playlists, and listening history.
- Extend audio store state with queue and resume metadata.
- Persist the new local-first models with safe sanitization.

## Out Of Scope

- Reader overflow UI
- Global mini-player chrome
- Synced library behavior
